# praxichnology/.github

This is the special `.github` repository for the [**praxichnology**](https://github.com/praxichnology) GitHub organization.

Its only job is to host the organization profile: [`profile/README.md`](profile/README.md) is what renders on the org landing page at **[github.com/praxichnology](https://github.com/praxichnology)**.

There's nothing else to see in this repo. For the field commons, start at the [organization profile](https://github.com/praxichnology), or go straight to [`docs`](https://github.com/praxichnology/docs) (orientation, citation guidance, guides) and [`preprint`](https://github.com/praxichnology/preprint) (the founding paper).
